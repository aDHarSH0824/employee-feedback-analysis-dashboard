# Employee Feedback Analysis & Action Recommendation Dashboard

## Overview
This project analyzes employee feedback data and converts raw survey responses into structured insights, trends, and action-oriented recommendations to support leadership decision-making.

## Problem Statement
Organizations collect large volumes of employee feedback, but leadership often lacks a structured and data-driven way to analyze feedback and identify priority improvement areas.

## Solution
This dashboard processes employee feedback using rule-based sentiment analysis and issue categorization, and presents insights through interactive visualizations and summaries.

## Tech Stack
- Python
- Pandas
- Streamlit
- Plotly

## Status
Project under active development.
